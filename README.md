## 我的用户管理系统

> vue+bootstrap

#### 如何运行：(前提是得先运行json-server服务，取得端口号为3000的数据接口)
1. 初始化项目：
```
npm install
```
2. 运行项目：
```
npm run dev
```
#### json-server的数据接口代码地址：
> https://github.com/boleming/user-json-server
